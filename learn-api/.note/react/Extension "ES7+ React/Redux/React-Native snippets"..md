# Extension "ES7+ React/Redux/React-Native snippets".
Các lệnh gõ nhanh (Shortcuts):

rfc: Tạo một Function Component (với export mặc định).

rfce: Tạo Function Component có kèm dòng export default.

rafce: Tạo Function Component dưới dạng Arrow Function (rất phổ biến hiện nay).

useS: Tự động tạo một dòng useState.

useE: Tự động tạo một khối useEffect.
